04

AI-Assisted Finance

Finance professionals who know how to work with AI move faster, catch more, and build better. These are real examples of how I use Claude as a co-pilot — not to replace judgement, but to sharpen it.

ZBB Sanity Check & Budgeting Suggestions

Instead of manually hunting for formula errors, I run a ZBB (Zero Based Budgeting) file through Claude for a full sanity check. It flags inconsistencies between assumptions, identifies gaps in cost coverage, spots formula logic issues, and suggests information I might be missing — all in seconds. What used to take hours of review is now a structured, repeatable quality gate.

Automated NetSuite → Cost Review Workflow

I built a custom Claude skill that automates the full end-to-end cost review update. When a new NetSuite file lands in a folder, Claude cleans the data (fills blank GL account names, resolves missing vendor references), replaces the source tab in the cost review workbook, and saves a new versioned file — fully hands-off. A process that previously required 20–30 minutes of careful manual work now runs in under a minute, with zero risk of human error.
